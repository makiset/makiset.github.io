HTML

HTML stands for Hyper Text Markup Language and is the standard markup language for creating web pages.

Example:

                                 <!DOCTYPE html>
                                 <html>
                                 <head>
                                 <title>Page Title</title>
                                 </head>
                                 <body>
                                    
                                 <h1>My First Heading</h1>
                                 <p>My first paragraph.</p>
                                    
                                 </body>
                                </html>
